Andromeda Theme
================

Andromeda Look & Feel package.

## Contents

* Color scheme
* Icons, based on Breeze https://quickgit.kde.org/?p=breeze-icons.git
* Plasma theme
* Sddm theme, lock screen and splash screen based on Breeze.
* Andromeda wallpaper
